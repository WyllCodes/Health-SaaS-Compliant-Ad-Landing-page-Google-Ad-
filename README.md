# About This Project
This Repository contains a demo health SaaS Google Ads campaign and landing page built to demonstrate:
   - Google Ads Health Policy Awareness
   - SaaS Marketing Strategy
   - Conversion-Focused Landing Page design

# Why This is Complaint
   - No medical claims
   - No patient data
   - No live advertising
   - Clear demo labeling

# Tools Used
  - Google Ads (Conceptual)
  - HTML/CSS (landing Page)
  - GitHub Pages (Hosting)
  


# Project Overview
#### Project Name: VamosHealth (Demo Health SaaS)
#### Project Type: Google Ads (Search) + Landing Page (Mock)

### Use Case: Portfolio demonstration of Google Ads policy-complaint Health SaaS Marketing

# Target Audience (B2B):
 - Clinics
 - Diagnostic Centers
 - Health Startups
 - Healthcare Administrators

# Core Value Proposition:
 VamosHealth helps healthcare organizations & Patients streamline operations, manage records, and improve workflow efficiency - without providing medical diagnosis or treatment.
